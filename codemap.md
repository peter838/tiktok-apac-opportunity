# Repository Atlas: DHL Singapore Opportunity

## Project Responsibility

Multi-country task management + analytics dashboard serving 9 APAC markets (AU, CN, HK, ID, IN, JP, MY, SG, TH). Two core systems:

1. **Task Tracker**: Country-scoped task management with per-market sequential IDs, status tracking, deadline management, and user access control.
2. **Analytics Dashboard**: Real-time aggregation of SourceSage API data — entity margins, order analysis, cost savings — displayed on root `index.html`.

Backend: Convex (realtime BaaS) with Supabase fallback for legacy API proxy.

## Country Coverage

All markets: AU (Australia), CN (China), HK (Hong Kong), ID (Indonesia), IN (India), JP (Japan), MY (Malaysia), SG (Singapore), TH (Thailand)

## Data Model

- **tasks**: per-country sequential ID, date, description, owner, deadline, status (not started / in progress / completed)
- **users**: email, name, role (admin/editor/viewer), status (active/inactive/pending), countries array, login tracking
- **countries**: static reference table of country codes and display names
